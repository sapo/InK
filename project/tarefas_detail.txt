INK
Notas e Definições

Todas as classes da framework devem ter um prefixo .ink-

Tables


Layouts:

	* Só com linhas nas rows .ink-lines
	* Grelha .ink-gridded
	* Zebra .ink-zebra
	* Com border na table .ink-bordered

Opções:

	* Hover nas rows .ink-rowhover
	* Pintar certas rows com uma class (vermelho, verde, azul) .ink-warning .ink-info .ink-caution .ink-success .ink-disabled

Forms

Layout de cada campo:

	* Label .ink-label
	* Help text .ink-helptxt
	* Campo/s
	* Error text .ink-warningtxt

Conjunto de controlos (checkboxes, radio buttons):

	* Usar fieldset legend para "label" do conjunto de controlos
	* Usar as labels em cada controlo

Botões > mostly done


Variantes:
	
	* Cinco "cores", usar .ink-warning, etc (ver acima)
	* Estados disabled para todos os anteriores (idem)
	* Três tamanhos .ink-medium .ink-small .ink-large (?)

Icons

	* Usar Font Awesome
	* Incluir CSS e webfont para ser chamada
	* Incluir attribution

Navegação

	* Navbar .ink-navbar
	* Navbar com dropdown .ink-dropdown
	* Navlist .ink-navlist
	* Navlist com flyout .ink-flyout
	* Sistema de navbar até três níveis inspirado no GLV

Mensagens e Avisos

	* Erro .ink-warning
	* Aviso .ink-caution
	* Informação .ink-info
	* Sucesso .ink-success
	* Espera .ink-wait
	* Disabled .ink-disabled

Layout dos avisos:

	* Título
	* Texto
	* Dismiss (cruz no canto)
	* Acções (nalguns casos) -> botões

Tipografia

Testar font "Ubuntu" e decidir se é para usar

	* Defnir todos os elementos
		* h1 a 6
		* p
		* ol, ul, dd
		* cite, blockquote
		* code
		* b, i, em, strong, mark
