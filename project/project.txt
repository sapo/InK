SAPO SUPER STUFF THINGY

Define e implementa uma linguagem visual "SAPO" nos principais sites e serviços do portal, recorrendo a três componentes principais e uma ferramenta.

Os componentes:

1. Ink - Interface Kit, um framework de HTML e CSS baseada em LESS
2. LibSAPO JS - Componentes essenciais a qualquer interface, human-usable
3. Look SAPO - Uma linguagem visual unificada que dá consistência aos serviços core (forms, botões, headers, barras)

A Ferramenta:

SAPO CE - Um configurador visual de layouts, com integração de componentes JS, que fornece um bundle com tudo o que o designer precisa para começar a codar as suas páginas/templates:

- Estrutura de directorias normalizada
- CSS base (com look sapo)
- HTML com o esqueleto do layout escolhido (headers, colunas, menus, etc)
- Chamadas aos componentes de JS necessários (carrossel, modal, tooltip, etc)
- Imagens (nomeadamente, uma webfont com icones facilmente implementáveis com recurso apenas a classes de CSS)
